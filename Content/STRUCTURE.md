Content folder layout (overview)
--------------------------------

- Content/Codex/Levels        : All Codex maps (e.g., Codex_Scifi_Landscape).
- Content/Codex/Materials     : Codex-specific materials and instances.
- Content/Codex/Meshes        : Imported/static meshes for Codex scenes.
- Content/Codex/Textures      : Textures and decals for Codex assets.
- Content/Codex/FX            : Niagara/VFX assets for Codex.
- Content/Codex/Audio         : Sound cues/wavs for Codex scenes.
- Content/Codex/UI            : Widgets/UMG for Codex tooling.
- Content/Codex/Prefabs       : Blueprint actors or prefabs used across scenes.
- Content/Codex/Showcase      : Overview/showcase assets for quick selection.

Existing runtime scripts: Content/Python (listener, menu, scene builders).
